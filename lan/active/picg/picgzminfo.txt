Description of the zoom function

Usage:

	- Select Zoom 2x or Zoom 4x in the View menu
	- Press the mouse button inside the drawing area
	- Move the rectangle to the desired position
	- Release the mouse button

	- Select Zoom 1x in the View menu to restore

Restrictions:

	- Only elements completely inside the selection
	rectangle will be zoomed
	- The zoomed elements will be moved to the topmost
	level of the drawing (The same as selecting the area
	and selecting Object/Raise)
